# CP6: Claude Code Defense Layer

Installs a PreToolUse hook that fires before every Write, Edit, or Bash call.
The hook scans content for hardcoded secrets and returns permissionDecision: "deny"
if found — Claude sees the reason and stops before writing the file.

## Files written

1. `.claude/hooks/save-yourself-check.sh` — hook script
2. `.claude/settings.json` — hook config (merged, not overwritten)

Both are project-level. Tip to show user:
"Add `.claude/settings.local.json` to `.gitignore` if you don't want hooks shared
with the team. Or commit `.claude/settings.json` to protect everyone."

## Step 1: Write hook script

```bash
mkdir -p .claude/hooks
```

Write `.claude/hooks/save-yourself-check.sh`:

```bash
#!/bin/bash
# save-yourself CC hook: blocks hardcoded secrets in Write/Edit/Bash calls
# Generated by /save-yourself — edit with care

INPUT=$(cat)
TOOL=$(echo "$INPUT" | python3 -c \
  "import sys,json; print(json.load(sys.stdin).get('tool_name',''))" 2>/dev/null)

case "$TOOL" in
  Write)
    CONTENT=$(echo "$INPUT" | python3 -c \
      "import sys,json; d=json.load(sys.stdin); \
       print(d.get('tool_input',{}).get('content',''))" 2>/dev/null)
    ;;
  Edit)
    CONTENT=$(echo "$INPUT" | python3 -c \
      "import sys,json; d=json.load(sys.stdin); \
       print(d.get('tool_input',{}).get('new_string',''))" 2>/dev/null)
    ;;
  Bash)
    CMD=$(echo "$INPUT" | python3 -c \
      "import sys,json; d=json.load(sys.stdin); \
       print(d.get('tool_input',{}).get('command',''))" 2>/dev/null)
    if echo "$CMD" | grep -qE '\.env[[:space:]]*(>|>>)'; then
      python3 -c "import json; print(json.dumps({
        'hookSpecificOutput': {
          'hookEventName': 'PreToolUse',
          'permissionDecision': 'deny',
          'permissionDecisionReason': 'Redirecting output to .env is blocked. Use .env.example instead.'
        }}))"
    fi
    exit 0
    ;;
  *)
    exit 0
    ;;
esac

MATCH=$(SAFE_CONTENT="$CONTENT" python3 - <<'PYEOF'
import re, os
content = os.environ.get('SAFE_CONTENT', '')
pattern = re.compile(r'(?i)(api[_-]?key|secret|token|password|passwd|pwd)\s*=\s*\S+')
placeholders = re.compile(r'(?i)(example|placeholder|changeme|your_|xxx|<|>|\$\{|\$\()')
matches = [m.group(0) for m in pattern.finditer(content)
           if not placeholders.search(m.group(0))]
safe = [(m[:40] + '...') if len(m) > 40 else m for m in matches[:3]]
print('\n'.join(safe))
PYEOF
)
# Note: SAFE_CONTENT env var avoids the pipe+heredoc stdin conflict (heredoc overrides
# pipe; content never reaches python3 stdin) and the bash injection risk.
# os.environ.get() is also immune to UnicodeDecodeError on binary content.

if [ -n "$MATCH" ]; then
  SAFE_MATCH=$(echo "$MATCH" | head -c 120)
  SAFE_MATCH="$SAFE_MATCH" python3 -c "
import json, os
match = os.environ.get('SAFE_MATCH', '').strip()
print(json.dumps({
  'hookSpecificOutput': {
    'hookEventName': 'PreToolUse',
    'permissionDecision': 'deny',
    'permissionDecisionReason': f'Hardcoded secret detected: {match}. Move to .env and reference via environment variable.'
  }
}))
"
  exit 0
fi

exit 0
```

Make executable:
```bash
chmod +x .claude/hooks/save-yourself-check.sh
```

## Step 2: Merge into .claude/settings.json

Use Python3 — safe to run twice (idempotent):

```python
python3 - <<'EOF'
import json, os

path = ".claude/settings.json"
try:
    with open(path) as f:
        settings = json.load(f)
except (FileNotFoundError, json.JSONDecodeError):
    settings = {}

hooks = settings.setdefault("hooks", {})
pre = hooks.setdefault("PreToolUse", [])

cmd = '"$CLAUDE_PROJECT_DIR"/.claude/hooks/save-yourself-check.sh'
already = any(
    any("save-yourself-check.sh" in h.get("command", "") for h in e.get("hooks", []))
    for e in pre
)

if not already:
    pre.append({
        "matcher": "Write|Edit|Bash",
        "hooks": [{
            "type": "command",
            "command": cmd,
            "statusMessage": "Scanning for secrets..."
        }]
    })
    os.makedirs(".claude", exist_ok=True)
    with open(path, "w") as f:
        json.dump(settings, f, indent=2)
    print("INSTALLED")
else:
    print("ALREADY_INSTALLED")
EOF
```

Verify post-install:
```bash
python3 -c "import json; s=json.load(open('.claude/settings.json')); \
  h=str(s.get('hooks',{})); print('CC Defense Layer: ACTIVE' if 'save-yourself-check.sh' in h else 'NOT INSTALLED')"
# Also confirm $CLAUDE_PROJECT_DIR path stored correctly:
grep CLAUDE_PROJECT_DIR .claude/settings.json
```

## Key implementation notes

- Exit code for blocking: return JSON with `permissionDecision: "deny"` + exit 0
  (NOT exit 2 — exit 2 skips JSON output; exit 0 + JSON gives Claude a structured reason)
- `python3` required for JSON parsing + regex. If unavailable, script exits 0 silently
  (non-blocking degradation — do not break the workflow)
- Same secret patterns as @references/secret-patterns.md for consistency
  (update both files if changing the regex pattern)
- Matcher `"Write|Edit|Bash"` — pipe-separated, exact match (not regex)
- `$CLAUDE_PROJECT_DIR` is set by Claude Code when running hooks (documented CC behavior).
  The hook path `"$CLAUDE_PROJECT_DIR"/.claude/hooks/save-yourself-check.sh` expands
  correctly at runtime.
